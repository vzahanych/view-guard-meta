# Encryption Libraries

End-to-end encryption libraries for The Private AI Guardian - client-side decryption tools for privacy-first video archiving.

## Overview

This is the single source of truth for all encryption primitives used across the platform:
- AES-256-GCM encryption
- Argon2id key derivation
- Browser-based decryption
- Edge encryption client

## Directory Structure

- `go/` - Go encryption library
- `typescript/` - Browser/Node.js library
- `python/` - Python library
- `cli/` - Optional CLI tool for decryption
- `docs/` - Encryption model and usage documentation
